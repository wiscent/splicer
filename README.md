# 📋 Image Slit Splicer Pro

> **Precision Multi-Slit Splicing Engine**
> 
> 이미지의 불필요한 구간을 슬릿(Slit) 기반의 폴딩(Folding) 기법으로 정밀하게 제거하고, 남은 영역을 자연스럽게 병합하는 이미지 편집 도구입니다.

## 1. 프로젝트 소개

**Image Slit Splicer Pro**는 단순한 이미지 크롭(Crop) 도구를 넘어, 이미지의 중간 영역을 '접어서' 제거하는 독특한 방식을 사용합니다. 웹툰, 긴 문서 스크린샷, 또는 불필요한 공백이 포함된 사진에서 특정 구간을 제거하고 상하 영역을 매끄럽게 이어 붙이는 작업에 최적화되어 있습니다.

## 2. 주요 기능 (Key Features)

### 🛠 다중 슬릿 관리 시스템

- **무제한 슬릿 추가:** 원하는 만큼 기준선(Slit)을 추가하여 이미지의 여러 구간을 동시에 편집할 수 있습니다.
  
- **고유 번호 유지:** 슬릿이 추가된 순서대로 번호가 부여되며, 중간에 슬릿을 삭제해도 기존 번호가 유지되어 혼선을 방지합니다.
  
- **Active 슬릿 강조:** 현재 조작 중인 슬릿은 시각적으로 강조되어 조작 대상을 명확히 알 수 있습니다.
  

### 🧠 지능형 배치 알고리즘 (Intelligent Placement)

- **1/3 자동 배치:** 새 슬릿을 추가할 때, 현재 선택된 슬릿과 아래 경계(다음 슬릿 또는 뷰포트 하단) 사이의 최적 지점(1/3 지점)을 자동으로 계산하여 배치합니다.
  
- **폴딩 영역 계산:** 단순히 원본 좌표가 아닌, 이미 폴딩(제거)된 영역을 제외한 실제 결과물 좌표를 기준으로 신규 슬릿 위치를 결정합니다.
  

### 📐 정밀 물리 엔진 (Visual Pinning)

- **비주얼 피닝:** 슬릿 상단 영역을 아래로 접을 때, 줄어드는 이미지 높이만큼 캔버스 위치를 역보정합니다. 이를 통해 사용자가 잡고 있는 슬릿 라인이 화면상에서 튀지 않고 고정되어, 이미지가 슬릿 안으로 '말려 들어가는' 듯한 직관적인 조작감을 제공합니다.

### 📱 완벽한 반응형 및 터치 지원

- **Auto-Fit:** 이미지 로드 시 뷰포트 너비에 맞춰 자동으로 확대/축소 배율을 조정합니다.
  
- **하이브리드 조작:** 마우스 드래그는 물론, 모바일에서의 핀치 줌(Pinch Zoom)과 터치 이동을 완벽하게 지원합니다.
  
- **통합 히트 테스트:** 얇은 슬릿 라인을 정확히 터치하지 않아도 반경 30px 이내의 터치를 감지하여 이동과 폴딩을 스마트하게 구분합니다.
  

## 3. 조작 가이드 (Interaction Guide)

| **액션** | **마우스 조작** | **터치 조작** |
| --- | --- | --- |
| **슬릿 이동** | 슬릿 라인 근처(30px) 드래그 | 슬릿 라인 근처 터치 후 드래그 |
| **상단 폴딩** | 슬릿 위쪽 이미지 영역 아래로 드래그 | 슬릿 위쪽 이미지 영역 아래로 드래그 |
| **하단 폴딩** | 슬릿 아래쪽 이미지 영역 위로 드래그 | 슬릿 아래쪽 이미지 영역 위로 드래그 |
| **화면 이동 (Pan)** | Space 키 + 드래그 또는 '이동' 모드 | '이동' 모드 활성화 후 드래그 |
| **확대/축소 (Zoom)** | 우측 하단 줌 버튼 | 두 손가락 핀치 줌 |
| **미리보기 (Hold)** | '미리보기' 버튼 클릭 유지 | '보기' 버튼 터치 유지 |

## 4. 기술 명세 (Technical Specs)

- **언어:** HTML5, CSS3, JavaScript (Vanilla JS)
  
- **스타일링:** Tailwind CSS
  
- **렌더링:** HTML5 Canvas 2D API (`drawImage`)
  
- **좌표 변환 로직:**
  
  - `Source` (원본 픽셀) ↔ `Canvas` (병합된 결과물) ↔ `Viewport` (화면 표시)
    
  - 폴딩 영역을 제외한 소스 세그먼트를 추출하여 픽셀 틈새 없이(`Math.round`) 렌더링합니다.
    

## 5. 시작하기

1. `image_slit_splicer_pro.html` 파일을 웹 브라우저로 엽니다.
  
2. **[이미지 열기]** 버튼을 통해 편집할 이미지를 불러옵니다.
  
3. 이미지 내의 슬릿 라인을 이동시켜 기준을 잡습니다.
  
4. 슬릿의 상/하단 영역을 드래그하여 불필요한 부분을 접어 제거합니다.
  
5. **[저장]** 버튼을 눌러 결과물을 PNG 파일로 내려받습니다.
  

*본 도구는 사용자 기기 내에서 모든 처리가 이루어지는 클라이언트 사이드 애플리케이션으로, 업로드한 이미지는 서버로 전송되지 않습니다.*
